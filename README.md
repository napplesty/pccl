## PCCL (Programable Communication & Commputation Library)
PCCL is a high performance communication library for parallel training. PCCL supports many features to boost the performance, including:
- **Runtime Resource Control**
- **Multiple Communication Protocol Support**
- **Flexible Backend Support**
- **Programmable Communication**
- **Communication-Computation Overlap**
- **Communication-Communication Overlap**
- **Heterogeneous Device Support**
